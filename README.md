## how to use

* include module block code

```code
module "users" {
  source = "./modules/users"
  username = "dddddemo"
  consul_host ="http://127.0.0.1:8500"
}
```

* get  local module

```code
terraform get
```

* init

```code
terraform  init
```

* view plan

```code
terraform plan
```

* apply

```code
terraform apply
```

* output result file

like below
